# notepad
Jot your ideas down with this cute lil notepad
![img](https://raw.githubusercontent.com/ThatZiv/notepad/master/media/Capture.PNG)

look how cute it is.

## Usage

`/notepad` toggles it

### *optional*

`/notepad open` opens it

`/notepad close` closes it
